# Industrial Image Defect Detection

This project develops a machine learning pipeline for detecting visual defects in industrial product images.

## Motivation

Industrial inspection often relies on image data to identify defective parts, surface damage, or production issues. This project explores image based anomaly detection as a practical AI/ML workflow for industrial quality control.

## Dataset

Planned dataset: MVTec AD industrial anomaly detection dataset.

The dataset will not be uploaded to this repository. Download instructions will be provided in `data/README.md`.

## Methods

Planned methods:

- Image preprocessing
- Baseline anomaly detection
- PatchCore or PaDiM anomaly detection
- Image-level evaluation
- Defect localization heatmaps
- Failure-case analysis
